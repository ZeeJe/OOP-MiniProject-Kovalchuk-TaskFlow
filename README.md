# TaskFlow (OOP Mini Project)
Цей проєкт створений в рамках Лабораторних робіт з об'єктно-орієнтованого програмування.

## Запуск програми
1. Відкрийте термінал у кореневій папці проєкту.
2. Виконайте команду: `dotnet run --project src/TaskFlow.Console`

## Запуск тестів
Виконайте команду: `dotnet test`
